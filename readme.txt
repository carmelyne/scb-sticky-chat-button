Add a customizable sticky chat Facebook Messenger button in a bubble style to your WordPress site, perfect for linking to messaging platforms or contact pages.

== Description ==

SCB - Sticky Chat Button allows you to add a sleek, customizable sticky chat button in a bubble style to your WordPress website. This button can link to any URL you choose, making it perfect for connecting visitors to your preferred messaging platform, contact page, or customer support system.

Features:

* Easy to install and configure
* Customizable button colors
* Responsive design
* Lightweight and fast-loading
* Compatible with all modern browsers
* Sticky positioning for constant visibility
* Bubble style similar to popular messaging apps

Whether you're looking to improve customer engagement, provide quick access to support, or simply add a convenient contact option, SCB - Sticky Chat Button is the perfect solution.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/scb-sticky-chat-button` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the Settings -> SCB - Sticky Chat Button screen to configure the plugin.

== Frequently Asked Questions ==

= Can I customize the appearance of the button? =

Yes, you can customize the background color of the button and the color of the chat icon through the plugin settings.

= Can I link the button to any URL? =

Absolutely! You can set the button to link to any URL you choose. This could be a messaging platform, a contact form, or any other page you prefer.

= Is the button always visible? =

Yes, the sticky chat button is designed to always be visible, typically in the bottom right corner of the screen. It will remain in place as the user scrolls the page.
